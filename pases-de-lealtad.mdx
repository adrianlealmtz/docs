# Diseño de Pases de Lealtad

Los Pases de Lealtad de Alara están diseñados para ser completamente **dinámicos**.

A diferencia de una tarjeta física o una imagen estática, el contenido del pase puede **cambiar** **automáticamente** conforme el usuario **avanza** dentro de su programa de lealtad.

Esto permite mostrar distintos diseños, recompensas, promociones o cambios de estatus **sin que el usuario** tenga que volver a descargar su pase.

<Info>
  Gracias a nuestra integración oficial con Apple Wallet y Google Wallet, Alara puede actualizar **automáticamente** el diseño y contenido del pase conforme ocurren eventos como la acumulación de sellos, cambio de nivel o desbloqueo de recompensas.
</Info>

---

# ¿Cómo funciona?

Cada Banner Image representa un estado diferente del pase.

Cuando el usuario obtiene un nuevo sello, Alara **actualiza automáticamente** el Banner Image correspondiente.

Por ejemplo, en una tarjeta de **10 sellos**, normalmente existirán **11 imágenes**.

| Imagen | Estado del pase |
| --- | --- |
| Imagen 0 | Tarjeta recién instalada (0 sellos) |
| Imagen 1 | 1 sello |
| Imagen 2 | 2 sellos |
| Imagen 3 | 3 sellos |
| ... | ... |
| Imagen 10 | 10 sellos / recompensa final |

De esta manera el usuario siempre visualiza el progreso real de su tarjeta.

# No siempre debe seguir un orden

Aunque normalmente existe una imagen para cada cantidad de sellos, esto **no es obligatorio**.

Puedes cambiar el Banner Image en **cualquier momento** mediante reglas o automatizaciones.

Algunos ejemplos:

- Mostrar un premio sorpresa.
- Mostrar un descuento desbloqueado.
- Mostrar un banner temporal por temporada.
- Mostrar un mensaje de cumpleaños.
- Mostrar un diseño especial para Navidad.
- Mostrar una promoción limitada.

El cambio del Banner Image depende completamente de la lógica que definas para tu programa.

<Tip>
  Muchos de nuestros clientes utilizan un Banner Image completamente distinto cuando el usuario **desbloquea** una recompensa. Esto genera una sensación de logro mucho mayor que simplemente agregar otro sello.
</Tip>

# Recompensas dinámicas

Una excelente práctica consiste en cambiar completamente el diseño del pase cuando el usuario obtiene una recompensa.

Por ejemplo:

- Los primeros cinco sellos muestran únicamente el **progreso**.
- Al llegar al sello 5 aparece un Banner Image **anunciando** el premio ganado.
- Después de canjear el premio, el pase **continúa** normalmente hacia la siguiente recompensa.

Esto hace que el programa sea mucho más atractivo para el usuario.

# Assets del Pase

Un Pase Digital está compuesto por varios Assets.

Cada uno tiene una función distinta dentro de Apple Wallet y Google Wallet.

---

# Banner Image

El Banner Image es la imagen **principal** del pase.

Se muestra en la parte superior de la tarjeta y normalmente representa el estado actual del programa de lealtad.

### Especificaciones

- Resolución recomendada: **3720 × 1476 px**
- Relación: **1860 × 738 px (3x recomendado)**
- Mantener siempre la misma proporción.

<Info>
  Todos los Banner Images de un mismo programa deben tener exactamente las **mismas** **dimensiones** para evitar cambios de tamaño entre estados.
</Info>

---

# Logo

El Logo aparece en la parte superior izquierda del pase.

Generalmente corresponde al logotipo de tu marca.

### Especificaciones

- Hasta **960 px** de ancho.
- Altura flexible.

Si el logotipo es demasiado ancho, Apple Wallet ocultará parte de la información del encabezado.

<Tip>
  Recomendamos utilizar versiones horizontales del logotipo con fondo transparente.
</Tip>

---

# Icon

El Icon es utilizado principalmente para:

- Notificaciones Push.
- Centro de notificaciones.
- Pantalla bloqueada.
- Algunas vistas previas del pase.

### Especificaciones

- Resolución mínima: **720 × 720 px**
- Relación: **1:1**
- Fondo transparente recomendado.

---

# Color del Pase

Además de las imágenes, también puedes modificar el color general del pase.

Esto permite diferenciar distintos niveles dentro de un mismo programa.

Por ejemplo:

- Bronce
- Silver
- Gold
- Platinum
- VIP

Muchas marcas utilizan distintos colores para comunicar visualmente el nivel del cliente.

## Paso 5
